#  Projet : Pendu Client-Serveur

Application de pendu Java fonctionnant en architecture client-serveur

---

## 📝 Présentation

Ce projet est un projet d'etudes dont l'objectif était de concevoir un jeu du pendu en Java client-serveur où il est possible de se connecter au  serveur pour demarrer une partie,via le client.

### Stack Technique
* **Langage** : Java
* **Framework/Lib** : JavaFX
* **Outils** : IntelliJ

---

## ⚙️ Installation

Suivez ces étapes pour mettre en place votre environnement de développement.

### Prérequis
* **Systeme d'exploitation** : Windows 10 ou 11
* **Logiciels** : n'importe quel IDE Java

### Étapes d'installation

1. **Clonez le dépôt.**
2. **Ouvrez votre IDE et chargez le projet.**
3. **Exécutez le projet.**
